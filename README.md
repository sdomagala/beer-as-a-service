# beer-as-a-service

What you need to start it up:
- Node.js(v4.0.0 or higher) and npm(v3.0.0 or higher) installed 
- then

```
cd path/to/repo
npm install
node index
```

or (to run it on 8 cores):

```
cd path/to/repo
npm install
node clusters
```

Performance tests (with server running):

```
npm run performance-test
```
