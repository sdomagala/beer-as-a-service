# beer-as-a-service

What you need to start it up:
- Node.js and npm installed 
-- Node - 4.0.0 or higher
-- npm - 3.0.0 or higher
- then

```
cd path/to/repo
npm install
node index
```

or (to run it on 8 cores):

```
cd path/to/repo
npm install
node clusters
```

Performance tests (with server running):

```
npm run performance-test
```
